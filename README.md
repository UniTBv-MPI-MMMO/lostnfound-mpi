# LostNFound - Campus Connect

> O platformă digitală centralizată pentru raportarea și recuperarea obiectelor pierdute în campus, eliminând haosul postărilor de pe rețelele sociale.

# 1. Descriere și Obiective
Aplicația rezolvă problema pierderii obiectelor personale în mediul universitar prin oferirea unui sistem structurat de gestiune a anunțurilor.
- **Obiectiv 1:** Reducerea timpului de recuperare a obiectelor prin filtrare pe categorii și locații.
- **Obiectiv 2:** Facilitarea contactului direct și sigur între cel care găsește un obiect și proprietar.
- **Public țintă:** Studenții și personalul administrativ al universității.
